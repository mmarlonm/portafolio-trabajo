# Web Portfolio Project

This is a web portfolio project built using React and Next.js. It showcases various projects and provides contact information.

## Project Structure

```
web-portfolio-nextjs
├── public
│   └── profile-image.jpg
├── src
│   ├── pages
│   │   ├── index.tsx
│   │   └── _app.tsx
│   ├── components
│   │   └── Header.tsx
│   ├── styles
│   │   └── globals.css
│   └── types
│       └── index.ts
├── package.json
├── tsconfig.json
└── README.md
```

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd web-portfolio-nextjs
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Run the development server:
   ```
   npm run dev
   ```

5. Open your browser and go to `http://localhost:3000` to view the portfolio.
